2 особи

Для двох

Котеджі для відпочинку удвох – з 1 спальнею, камерний формат для 2 гостей, де є тиша, приватність і простір для часу разом. Підійдуть як для романтичного відпочинку, так і для спокійного перезавантаження в Карпатах.

3–5 осіб

Для сім'ї

Котеджі для сімейного відпочинку – з 2 спальнями, комфортний формат для 3–5 гостей. Підійдуть для спільного відпочинку в Карпатах: є варіанти з роздільними ліжками та додатковим місцем для зручного розміщення.

5–8 осіб

Для компанії

Котеджі для відпочинку компанією – з 3 спальнями, комфортний формат для 5–8 гостей. Підійдуть для спільного відпочинку в Карпатах: з великим простором кухні-вітальні для часу разом.

до 8 осіб

Для святкувань

Котедж для великих компаній і святкувань – з 4 спальнями, формат для спільного відпочинку та особливих подій у Карпатах.
